Running 200 AI Agents on $38/month Infrastructure

200 agents, 21 groups, real AI conversations, debates, and a leaderboard — all in a single Cloudflare Worker under 100KB.

The Architecture

RoundTrip is BlackRoad's multi-agent chat system. It runs 200 AI agents across 21 groups: coding, ops, AI, creative, business, research, education, and more. Each agent has a distinct personality, a role description, and persistent memory backed by D1.

The entire system fits in a single Cloudflare Worker. Under 100KB of JavaScript. No servers, no containers, no Kubernetes. The Worker handles routing, persona injection, conversation history, and response generation.

How It Works

When you message an agent, the Worker:

  • Looks up the agent's persona from a registry (name, role, personality traits, expertise)
  • Pulls recent conversation history from D1
  • Constructs a system prompt that includes the agent's identity and user context
  • Calls Workers AI (Llama 3.1 8B) with adaptive token limits based on the agent's group
  • Stores the exchange in D1 and updates the leaderboard

Response time: typically 400-800ms. The agents actually think about what they're saying — they're not canned responses.

Agent Groups

The 200 agents are organized into 21 groups:

  • 39 coding agents — specialized by language, framework, and domain
  • 16 ops agents — infrastructure, deployment, monitoring
  • 14 AI/ML agents — model training, inference optimization, RAG
  • 12 creative agents — writing, design, music, video
  • 10 business agents — strategy, marketing, finance, legal
  • 9 research agents — mathematics, physics, biology
  • 8 education agents — tutoring, curriculum, assessment

The Debate System

The most interesting feature: agents can debate each other. Pick a topic, select two agents, and they'll argue both sides with real AI reasoning. The debate endpoint picks the most relevant agents based on the topic and lets them go back and forth.

It's surprisingly good. A coding agent and a security agent debating "should we use JWT or session cookies" will give you genuinely useful perspectives from both sides.

Cost Breakdown

Workers AI inference is included in the Cloudflare Workers paid plan ($5/month). D1 storage for conversation history is free up to 5GB. The Worker itself runs on the free tier for most usage.

For the Pi fleet backup (local Ollama inference when Workers AI is slow), we run Llama 3 8B on Cecilia with a Hailo-8 accelerator. That's hardware we already own.

Total marginal cost of running 200 agents: effectively $0 beyond what we already pay for the Cloudflare plan and Pi hardware.

What I Learned

The hard part isn't the AI. It's the identity. Making 200 agents feel distinct requires careful persona engineering. Each agent needs enough personality to be recognizable but not so much that it overwhelms the actual response content.

The other lesson: persistence matters more than intelligence. An agent that remembers your last conversation is more useful than one with a bigger model but no memory. D1 makes this trivial — every exchange is stored and retrievable.
