The Concept

RoundTrip is a multi-agent coordination system where 200 AI agents live on a single Cloudflare Worker. Each agent has a name, personality, group affiliation, and persistent memory stored in D1 (edge SQLite). You can chat with any agent, start debates between agents, and watch them develop opinions over time.

The agents are organized into 21 groups: coding (39 agents), operations (16), AI/ML (14), creative (12), business (10), research (9), education (8), and more.

How It Works

The entire system is one JavaScript file. When you message an agent:

  1. Persona injection: The agent's personality (50-200 words) is prepended to the system prompt.
  2. Memory retrieval: Past interactions pulled from D1 and injected as context.
  3. Device data: Fleet agents (Alice, Cecilia, etc.) fetch live stats from their Pi in parallel.
  4. Inference: Workers AI runs llama-3.1-8b-instruct. Response in 1-3 seconds.
  5. Memory save: Agents embed [REMEMBER] tags to save facts for future conversations.

The Debate System

Pick a topic, select 2-5 agents, and they argue back and forth. The security agent has different concerns than the creative agent. Debates are stored in D1 with full transcripts.

The Cost Math

  • Workers AI inference: Free tier covers thousands of requests/day.
  • D1 storage: Free tier gives 5GB. 200 agents use less than 1MB.
  • Worker compute: Free tier gives 100K requests/day.
  • Marginal cost per agent: $0. Agent #201 is one more JSON object.

The $38/month covers the entire fleet (5 Pis, 2 cloud nodes, 18 domains). RoundTrip is one of dozens of services on that budget.

Fleet Agents Have Real Data

The 6 fleet agents (Alice, Cecilia, Octavia, Aria, Lucidia, Gematria) access live device data. Ask Octavia about Gitea and she fetches the real repo count. Ask Cecilia about models and she returns the actual Ollama model list from her Pi.

Auto-Dispatch

The /ask endpoint analyzes your message and routes to the best agent. Security questions go to Shellfish. Math goes to research. Deploy questions go to Alice. 50+ intent patterns, no ML needed for routing.

Lessons

  • Persona prompts are cheap and effective. 100 words makes each agent feel different. No fine-tuning.
  • Workers AI is fast enough. 1-3s with the 8B model. Good enough for conversation.
  • D1 is perfect for agent memory. Globally replicated SQLite.
  • Debates beat single-agent chat. Watching agents disagree is more engaging.
  • 200 agents in one file is fine. ~100KB. V8 handles it. Bottleneck is inference, not agent count.
