Setup Guide

Deploy a BlackRoad Node

From bare metal to a fully connected fleet node in six steps. Every BlackRoad node is sovereign -- your hardware, your data, your rules.

1

Get a Raspberry Pi

Raspberry Pi 5 (8 GB) is recommended for best performance. Pi 4 (4 GB+) works for lighter workloads. Any ARM64 SBC with Debian support will work.

# Recommended hardware Raspberry Pi 5 (8 GB) # primary recommendation Raspberry Pi 4 (4 GB+) # budget option NVMe HAT + 256 GB SSD # required for fleet nodes USB-C power supply (27W) # official Pi 5 PSU
2

Flash Raspberry Pi OS

Use the Raspberry Pi Imager to flash Raspberry Pi OS Lite (64-bit, Bookworm). Lite is preferred -- no desktop overhead means more resources for your agents.

# Download Raspberry Pi Imager # https://www.raspberrypi.com/software/ # Or flash from the command line (macOS/Linux) $ wget https://downloads.raspberrypi.com/raspios_lite_arm64/images/raspios_lite_arm64-2024-11-19/2024-11-19-raspios-bookworm-arm64-lite.img.xz $ xz -d 2024-11-19-raspios-bookworm-arm64-lite.img.xz $ sudo dd if=2024-11-19-raspios-bookworm-arm64-lite.img of=/dev/sdX bs=4M status=progress
3

Enable SSH and Set Hostname

Before booting, configure SSH access and a hostname. If you used the Imager, you can set these in the advanced options. Otherwise, create the config files on the boot partition.

# Enable SSH (create empty file on boot partition) $ touch /Volumes/bootfs/ssh # Set hostname after first boot $ ssh pi@raspberrypi.local pi@raspberrypi:~$ sudo hostnamectl set-hostname mynode pi@mynode:~$ sudo nano /etc/hosts # Replace "raspberrypi" with "mynode" # Set a strong password pi@mynode:~$ passwd # (Optional) Add your SSH key for passwordless login $ ssh-copy-id pi@mynode.local
4

Install BlackRoad OS

The install script handles everything: system packages, agent runtime, WireGuard config, fleet registration, and the BlackRoad CLI. Takes about 5 minutes on a fresh Pi.

pi@mynode:~$ curl -sSL https://blackroad.io/install | bash BlackRoad OS Installer v2.1 -------------------------- [1/6] Updating system packages... [2/6] Installing dependencies (git, python3, wireguard, jq)... [3/6] Cloning blackroad-operator... [4/6] Configuring WireGuard mesh... [5/6] Registering node with fleet... [6/6] Starting agent runtime... BlackRoad OS installed. Run `br status` to verify.
5

Connect to Fleet via WireGuard

The installer generates a WireGuard keypair and requests a peer config from the fleet gateway. Once connected, your node can reach every other node on the mesh.

# Check WireGuard status pi@mynode:~$ sudo wg show interface: wg0 public key: aBcDeFgHiJkLmNoPqRsTuVwXyZ... listening port: 51820 peer: xYzAbCdEfGhIjKlMnOpQrStUvW... endpoint: gateway.blackroad.io:51820 latest handshake: 4 seconds ago transfer: 1.24 MiB received, 892 KiB sent # Verify fleet connectivity pi@mynode:~$ br fleet ping alice 10.10.0.49 2ms online cecilia 10.10.0.96 3ms online octavia 10.10.0.101 2ms online mynode 10.10.0.200 0ms local
6

Deploy Your First Agent

Use the BlackRoad CLI to spawn an agent on your new node. Agents run as lightweight processes, coordinated through the fleet mesh.

# Check node status pi@mynode:~$ br status Node: mynode Fleet: connected (4 peers) Agents: 0 running Uptime: 12 minutes Memory: 412 MB / 8192 MB # Deploy a monitor agent pi@mynode:~$ br agent spawn monitor Spawned agent "monitor" (pid 4821) Reporting to fleet dashboard... # List running agents pi@mynode:~$ br agent list ID NAME STATUS UPTIME a-001 monitor running 30s # View agent logs pi@mynode:~$ br agent logs monitor [2026-03-21 14:22:01] Monitor agent started [2026-03-21 14:22:02] CPU: 4.2% | Mem: 5.0% | Temp: 48C [2026-03-21 14:22:32] CPU: 3.8% | Mem: 5.1% | Temp: 47C

Your node is live. Welcome to the fleet.

Recommended Hardware

Raspberry Pi 5 (8 GB)
Quad-core Cortex-A76 @ 2.4 GHz. Best performance for multi-agent workloads and local inference.
recommended
Hailo-8 M.2 Module
26 TOPS neural accelerator. Enables on-device AI inference without GPU. Plugs into Pi 5 M.2 HAT.
ai accelerator
NVMe SSD (256 GB+)
10x faster than SD card. Required for Gitea, databases, and agent state. Use with Pi 5 M.2 HAT or USB 3.0 enclosure.
storage
Raspberry Pi 4 (4 GB)
Quad-core Cortex-A72 @ 1.8 GHz. Suitable for lightweight agents, DNS, or monitoring nodes.
budget
PoE+ HAT
Power over Ethernet. Single cable for power and data. Clean deployments, no wall adapter needed.
optional

Checklist

  • Raspberry Pi with power supply
  • NVMe or high-endurance SD card
  • Ethernet cable or Wi-Fi configured
  • Raspberry Pi OS Lite flashed
  • SSH enabled
  • BlackRoad OS installed
  • WireGuard connected
  • First agent deployed
